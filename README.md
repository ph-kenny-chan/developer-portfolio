# Developer Portfolio

A stylish and customizable portfolio template for developers to showcase their skills, projects, and experiences.

![developer-portfolio](https://github.com/shaqdeff/Portfolio-Template/assets/92020713/e41ed31c-50e5-40d3-a74f-9c0eb053fe7f)
![developer-portfolio](https://github.com/shaqdeff/Portfolio-Template/assets/92020713/e8c23aa7-ac27-48fc-bf66-50ab00e3a64f)

## Overview
This portfolio template is designed using modern web technologies and tools, enabling developers to present their work professionally. Feel free to use it, modify it, and make it your own. 

Projects and experiences showcased in this portfolio reflect personal efforts and creativity. Kindly avoid misrepresenting them as your own.

This project was initially **forked from [Shaq's Portfolio Template](https://github.com/shaqdeff/Portfolio-Template).** Full credit to the original author for creating the foundation.

## Features
- **Built With:**
  - React.js
  - Three.js
  - Framer Motion
  - TailwindCSS

- **Key Functionalities:**
  - Interactive animations powered by Framer Motion and Three.js.
  - Mobile-responsive design optimized with TailwindCSS.
  - Easy deployment for showcasing your projects.

## Getting Started

Follow these steps to set up and customize the portfolio:

### Prerequisites
- Node.js (`16.0.0` or higher)
- npm (`9.0.0` or higher)

### Setup
Clone the repository, or download the ZIP file:
```bash
git clone https://github.com/ph-kenny-chan/developer-portfolio.git
cd developer-portfolio
```

Open the project in [Visual Studio Code](https://code.visualstudio.com/download):
```bash
code .
```

### Installation
Install project dependencies using:
```bash
npm install
```

### Usage
Run the development server locally:
```bash
npm run dev
```
Access the portfolio at `http://localhost:5173/`.

### Build
For production, generate a build:
```bash
npm run build
```

To preview the production build locally:
```bash
npm run preview
```
